Troubleshooting

Solutions for common issues with Snap for Harvest.

Snap for Harvest Button Not Appearing

For Google Calendar or Notion

  1. Check that the integration is enabled in Options → Default Sites
  2. Refresh the page (Cmd/Ctrl + R)
  3. Make sure you're signed into your Harvest account
  4. Try disabling and re-enabling the integration

For Custom Sites

  1. Verify the URL pattern matches the current page
  2. Check that the site is enabled in Options → Custom Sites
  3. Refresh the page after adding the site
  4. Try removing and re-adding the site

Timer Not Loading

  • Make sure you're signed into your Harvest account
  • Check your internet connection
  • Try signing out and back in through the extension
  • Clear your browser cache and reload

Page Title Not Auto-Filling

  • This is expected on some sites with dynamic content
  • You can manually edit the Notes field in the Harvest timer
  • For custom sites, the page title should populate automatically
  • Try refreshing the page and clicking the button again

Extension Not Working After Update

  1. Close and reopen your browser
  2. Disable and re-enable the extension
  3. Check for Chrome browser updates
  4. Reinstall the extension if issues persist

Still Need Help?

If you're still experiencing issues, please contact us:

  • Email: support@snapharvest.xyz
  • Include: Browser version, extension version, and steps to reproduce the issue
